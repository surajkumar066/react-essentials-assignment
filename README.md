# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some Oxlint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the Oxlint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and Oxlint's TypeScript related rules in your project.

# React Essentials Assignment

This repository contains two mini React projects built as part of the React Essentials module, covering core concepts like components, props, state management, event handling, and conditional rendering.

## Part A: Portfolio Card App

A personal portfolio card built using reusable React components.

**Features:**
- Profile card displaying name, title, bio, and photo
- Skills rendered dynamically from props using `.map()`
- Theme switcher (light/dark mode) using React state
- Profile photo cycling on button click
- Alert popup on button click
- Like counter that updates on click using `useState`

**Concepts covered:** JSX, component structure, props, event handling, state management

**Folder:** `PartA-PortfolioCard`

## Part B: Movie Database App

A movie search and favourites app using local data (no external API).

**Features:**
- Search bar with live filtering of movies by title
- Conditional rendering for three states: no input, no results found, and results found
- Favourite/unfavourite toggle on each movie card
- Separate "Favourite Movies" section showing all favourited movies

**Concepts covered:** controlled inputs, array filtering, conditional rendering, state management with arrays

**Folder:** `PartB-MovieDatabase`

## Tech Stack
- React (Vite)
- Plain CSS

## How to Run Locally
```bash
npm install
npm run dev
```
